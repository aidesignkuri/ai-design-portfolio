## 0. 全体コンセプト

> **「読むだけで、LifeCreation流のLINE運用が体験できる公式LINE」**

このLINE自体が、整体院・サロンオーナーへの「もしうちで作ったらこういう体験を提供できますよ」という**生きたデモンストレーション**になる。説明より体験で売る、サイトの「静かな工房」トーンとそのまま地続き。

| 項目 | 設定 |
|---|---|
| ターゲット | 整体院・サロンオーナー（個人事業〜小規模） |
| 主ゴール | 「無料Webチェックシート」申込 |
| サブゴール | 長期関係構築・自社サービス事例化 |
| 配信戦略 | 1週間ステップ配信（5通）+ 月1-2回の通常配信 |
| トーン | サイトと同じ「静かな工房」、押し売りしない |

## 11. お店のWeb現状チェックシート 設計（コンバージョンゴール）

### コンセプト

> サイトの既存フレーム「**知ってもらう → 来てもらう → また来てもらう**」をそのまま3ブロックに使う。

診断結果がそのままLifeCreationの3サービス（HP制作・MEO・LINE運用）の必要性に対応する構造。診断しながら「自分に何が足りないか」が自然にサービスとつながる。

| 設定 | 内容 |
|---|---|
| 形式 | Googleフォーム（**クイズモード**でスコア自動表示） |
| 項目数 | 9問（3フェーズ × 3問） |
| 所要時間 | 約3分 |
| 採点 | 各問 2点 / 1点 / 0点（満点18点） |
| 診断 | スコア帯で3タイプに自動分類 |
| ハイブリッド | 末尾で「店名・URL」任意記入 → 希望者に栗本が個別コメント |

### 9つのチェック項目

#### ブロック1：知ってもらう（見つけてもらえているか）

| # | 質問 | できている(2) / あいまい(1) / できていない(0) |
|---|---|---|
| 1 | スマホで見やすいホームページがある | ✓ / 一応あるが古い / ない |
| 2 | Googleマップ（ビジネスプロフィール）に登録している | ✓ / 登録だけ / していない・不明 |
| 3 | 「地域名＋整体（サロン）」で検索すると自店が出てくる | ✓ / たまに / 出ない・不明 |

#### ブロック2：来てもらう（行きたいと思わせているか）

| # | 質問 | できている(2) / あいまい(1) / できていない(0) |
|---|---|---|
| 4 | Googleマップに写真が10枚以上ある | ✓ / 数枚 / ほぼない |
| 5 | 口コミが10件以上あり、返信もしている | ✓ / 口コミはあるが返信なし / ほぼない |
| 6 | 電話以外でも予約・問い合わせができる（ネット予約・LINE等） | ✓ / 一部 / 電話のみ |

#### ブロック3：また来てもらう（再来店の仕組みがあるか）

| # | 質問 | できている(2) / あいまい(1) / できていない(0) |
|---|---|---|
| 7 | 公式LINEなど、再来店を促す連絡手段がある | ✓ / あるが使えていない / ない |
| 8 | 来店後の次回予約やフォローの仕組みがある | ✓ / 声かけ程度 / 特にない |
| 9 | 新メニューやキャンペーンを定期的に発信している | ✓ / 気が向いたとき / していない |

### スコア診断（3タイプ）

| スコア | タイプ | 診断メッセージの方向性 | 誘導サービス |
|---|---|---|---|
| **14-18点** | 🌳 土台しっかり型 | 「基礎は整っています。あとは"続ける仕組み"で差がつきます」 | LINE運用 |
| **8-13点** | 🌱 伸びしろ型 | 「半分はできています。弱いブロックを1つ埋めるだけで変わります」 | 弱点ブロック対応サービス |
| **0-7点** | 🌰 これから型 | 「まだ伸びしろだらけ＝チャンス。まず『見つけてもらう』から」 | HP制作 / MEO |

> Googleフォームを**クイズモード**にすると、回答直後にスコアが自動表示される。サンクス画面に3タイプの解説を全部載せておけば、見込み客は自分のスコアでタイプを自己判定できる（フォームの分岐機能を使わずに済む＝制作が簡単）。

### ハイブリッドの「個別コメント」導線（フォーム末尾の任意項目）

```
─────────────────────────────
【ご希望の方限定・無料】

ご記入いただいた方には、栗本が実際に
あなたのお店のWebを拝見して、
「ここだけ直すと変わりますよ」というポイントを1つ、
LINEでお送りします。

▢ 店舗名（任意）：[          ]
▢ ホームページ or GoogleマップのURL（任意）：[          ]

※ 営業のご連絡はしません。気づいた点を1つお伝えするだけです。
─────────────────────────────
```

> サイトの「営業しない」トーンを最後まで貫く。これが他社との決定的な差別化になる。

### 運用フロー

```
[Day5でフォームURL送付]
   ↓
[見込み客がフォーム回答（3分）]
   ↓
[スコア自動表示 → 自分のタイプを知る]
   ↓
   ├─ URL記入あり → 栗本が手動でWeb確認 → LINEで個別コメント（質の高い接点）
   └─ URL記入なし → 通常配信へ（接点維持）
```

### 実装メモ

- Googleフォームの設定 → 「テスト（クイズ）」をオンにし、各選択肢に得点（2/1/0）を割り当てる
- 回答後に合計点を表示する設定をオンにする
- 確認メッセージ（サンクス画面）に3タイプの診断文をすべて記載
- フォーム回答はスプレッドシートに自動集計 → タグ付け・フォローに活用
- フォームURLは短縮 or LINE内ボタンに紐付け
