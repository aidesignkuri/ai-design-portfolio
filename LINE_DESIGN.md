## 0. 全体コンセプト

> **「読むだけで、LifeCreation流のLINE運用が体験できる公式LINE」**

このLINE自体が、整体院・サロンオーナーへの「もしうちで作ったらこういう体験を提供できますよ」という**生きたデモンストレーション**になる。説明より体験で売る、サイトの「静かな工房」トーンとそのまま地続き。

| 項目 | 設定 |
|---|---|
| ターゲット | 整体院・サロンオーナー（個人事業〜小規模） |
| 主ゴール | 「無料Webチェックシート」申込 |
| サブゴール | 長期関係構築・自社サービス事例化 |
| 配信戦略 | 1週間ステップ配信（5通）+ 月1-2回の通常配信 |
| トーン | サイトと同じ「静かな工房」、押し売りしない |

## 1. 友だち追加の導線

### 設置場所

| 場所 | 既存 | 状況 |
|---|---|---|
| サイト「LINEで相談する」CTA（コンタクトセクション右） | あり | 既設置 |
| サイトヘッダーCTA | なし | 検討余地 |
| Googleビジネスプロフィール | なし | 後日提案 |
| 名刺・チラシのQRコード | なし | 制作物次第 |

### 流入経路の動機付け

サイトのLINEセクションで、「**5日間で読める『お店のWeb現状チェック』ガイドをお届け**」とミニコピーを足すと、追加率が体感1.5-2倍変わる想定。実装する際は別途検討する。

## 2. 挨拶メッセージ（追加直後・即時送信）

```
LifeCreationの公式LINEへようこそ。
新潟県長岡市で、整体院・サロンの「お店づくり」を
お手伝いしているデザイン工房です。

このLINEでは、これから5日間にわたって、
「お客さまに、もう一度来てもらうためのWeb設計」
を1日1通お届けします。

押し売りや、しつこいご連絡は一切しません。
読んで、ご自身のお店のことを考える時間にしてもらえたら嬉しいです。

5日後には、ご希望の方へ
「お店のWeb現状チェックシート」をお送りします。

それでは、明日からよろしくお願いいたします。
                                 — LifeCreation
```

**設計意図**
- 期待値を最初に伝える
- 配信頻度を予告（毎日1通、5日間）
- ゴール（チェックシート）を予告
- → 離脱を防ぐ

## 3. ステップ配信 5通（1週間）

### 全体構成

| Day | テーマ | 役割 |
|---|---|---|
| **Day 1** | はじめまして — LifeCreationが大事にしていること | 信頼の土台 |
| **Day 2** | 「ホームページがあるのに、お客さまが来ない」のからくり | 共感・気づき |
| **Day 3** | 事例：リカバリー整体SORAでつくった「来てもらう仕組み」 | 実績で証明 |
| **Day 4** | 今あなたが読んでいる、このLINEの正体 | メタ気づき・デモ完成 |
| **Day 5** | あなたのお店のWeb、現状を一緒にチェックしませんか | コンバージョン |

### Day 1 本文サンプル（雰囲気の参考）

```
こんにちは、LifeCreationです。

私たちは、新潟県長岡市で
「お店づくり」のお手伝いをしている小さな工房です。

ホームページもMEOも公式LINEも、
すべてひとりで設計し、ひとりで仕上げます。

そして、ひとつだけ守っていることがあります。

「営業の電話をかけない、売り込みのメールも出さない」

これは、私自身がそういう連絡を受けるのが苦手だから、
というだけの理由ですが、お客さまにとっても
ちょうど良い距離感だと感じています。

明日は、「ホームページがあるのに、お客さまが来ない」
という、よく聞く悩みの正体について書きます。

— LifeCreation
```

> 各通の本文は別ステップでフルサイズで執筆する。現在は構成と長さ感のみ確定。

### Day 5 でのチェックシート提示の例

```
（前略：チェックシートの内容紹介）

▼ チェックシートをご希望の方はこちら
[ ボタン: 無料Webチェックシートをもらう ]

「いまの私のお店、本当にWebが効いているのかな」
そんなふうに思ったときに、お役立てください。

— LifeCreation
```

## 4. リッチメニュー 6マス設計

```
┌────────────────┬────────────────┬────────────────┐
│  📋             │  💡            │  🏪            │
│ 無料Webチェック  │ 最新の事例を見る │ サービス一覧   │
│  (メインCTA)    │ (chiropractic)│ (services/)   │
├────────────────┼────────────────┼────────────────┤
│  💬             │  📅            │  ℹ️            │
│ よくあるご質問  │  個別相談を予約  │  会社概要     │
│  (FAQ)          │  (zoom/対面)   │  (company/)   │
└────────────────┴────────────────┴────────────────┘
```

| マス | 行動 | リンク先 |
|---|---|---|
| 左上 ⭐ | 無料Webチェックシート申込 | LINE内フォーム or 外部フォーム |
| 中上 | 制作事例 | `/chiropractic/` `/middleex/` |
| 右上 | サービス一覧 | `/services/` |
| 左下 | よくあるご質問 | サイト #faq セクション |
| 中下 | 個別相談予約 | サイト #contact or 予約ツール |
| 右下 | 会社概要 | `/company/` |

> 左上は「メインCTA」として、色や絵文字で他5マスより目立たせる。

## 6. ステップ配信後の通常配信プラン

| 項目 | 設定 |
|---|---|
| 頻度 | 月1-2回 |
| 配信曜日 | 火曜 or 金曜の午前中（小規模事業者が落ち着いて読める時間） |

### コンテンツの引き出し（ローテーション）

1. **季節ネタ** — 「年末は新規HPの相談が増える時期です」「春は新メニュー写真の入れ替え時期」
2. **業界Tips** — 「Googleマップの口コミ返信、3つのコツ」
3. **新規事例UPDATE** — 新しい制作実績の紹介
4. **裏側エッセイ** — 「最近、ある整体院さんから相談を受けて気づいたこと」（押し付けない、エッセイ調）
5. **季節のあいさつ** — 年始・お盆など、形式的でない柔らかい挨拶

## 7. タグ・セグメント設計

| カテゴリ | タグ | 用途 |
|---|---|---|
| **業種** | `整体院` `サロン` `その他` | 将来の業種別配信に備える |
| **段階** | `ステップ中` `ステップ完了` `診断申込済` `相談予約済` `契約済` | コンバージョンファネル管理 |
| **興味** | `HP関心` `MEO関心` `LINE関心` | リッチメニューの反応で自動付与 |
| **流入元** | `サイト` `名刺` `紹介` `Googleマップ` | チャネル別効果測定 |

> LINE公式アカウントマネージャーの標準機能（「タグ」「リッチメニューのアクション計測」）で対応可能。

## 9. KPI・運用指標

| 指標 | 目標感（初期） |
|---|---|
| **友だち追加数** | 月3-5人 |
| **ステップ配信完走率** | 60%以上（Day 5まで読まれる） |
| **チェックシート申込率** | 追加者の20-30% |
| **チェックシート→個別相談率** | 申込者の50%以上 |
| **個別相談→契約率** | 30-50% |

→ **月3人追加 × 1人相談 = 年3-5件契約**、というベース感

## 10. 次の検討事項（v1.1以降）

### A. Webチェックシートの中身設計
ゴールに据えた「無料Webチェックシート」自体の設計。

- **形式**: PDF / Googleフォーム / Notion公開ページ / 個別カスタム回答
- **項目案**: ホームページの有無 → Googleマップ登録状況 → 口コミ数 → 公式LINE有無 → SNS運用 → 写真の質
- **納品形態**: その場で結果表示 / 数日後に手作業で返信 / 自動診断

### B. ステップ5通の本文フル執筆
各通約400-600字想定。サイトのトーンと合わせる。

### C. リッチメニュー画像のデザイン
6マス分のアイコン + テキストレイアウト。サイトと同じカラーパレットで。

### D. サイト側の改修
- LINEセクションのコピー強化（「5日間で読める...」のミニコピー追加）
- ヘッダーCTAへのLINEボタン追加可否
